1. Найдём кол-во страниц сайта с сериалами
2. Сформируем документ id включающий (genre, authors, actors, statuses, channels)
    2.0  Если документ id уже есть, то 3 пункт опускается
	2.1. Найдём количество сериалов
	2.2. Будём переходить по "https://www.toramp.com/schedule.php?page=" + (сериал по счёту) и собирать информацию :
           - о жанрах
           - об авторах
           - об актёрах
           - о возможных статусах сериалов
           - о каналах на которых сериалы выходят
	2.3. Запишем всё в глобальную переменную ID
3. Составим список url страниц каталога
4. Запустим multiprocessing
   Каждый  процесс обрабатывает свою страницу каталога :
    4.1  Получаем ссылки на все сериалы со страницы  
    4.2  Собираем информацию о сериалах,переходя по сссылкам :
	     4.2.1 О сериале :
           - ссылка
           - название
           - оригинальное название
           - дата выхода
           - краткое описание
           - описание
           - ссылка на картинку
           - имя картинки
           - id жанров
           - id статуса
           - id	каналов
		 4.2.2 Сезоны сериала :
		   - название
		 4.2.3 Эпизоды сериала :
		   - номер
		   - название
		   - оригинальное название
		   - дата выхода
		 4.2.4 Загружаем постер сериала
5. Индексация полученного содержимого и запись в json формат файл
